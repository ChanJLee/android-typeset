# API

##  约定

1. 所有以@Hidden标注的API都是不稳定API不要尝试调用

2. 不要尝试长期持有任何texas内部的数据结构，即用即走。

3. 拥有相同点击事件的元素被视为同一组元素，比如多个分割的text如果拥有相同的点击事件，那么在被点击的时候拥有相同回调的视图都会被高亮

4. 尽量避免多次调用TexasView的refresh操作，建议使用批量操作一次提交多个修改。

5. 如何区分有些属性应该库提供还是上层业务提供？根据属性的特征来算，如果跟业务无关，那么可以作为库应该提供的属性，比如绘制的字体颜色。但是像阅读某些单词应该有下划线，下划线的颜色还有两种，这样的就属于业务属性了。

6. 所有接口都可以通过当前的链接查找到[入口](java/index.html)
