## 🔍 符号数据结构

每个符号包含以下信息：

```json
{
  "unicode": "0x3b1",        // Unicode 编码
  "char": "α",               // 字符本身
  "glyphId": 4178,           // 字体中的字形ID
  "glyphName": "alpha",      // 字形名称
  "variants": [],            // 可拉伸变体（如有）
  "parts": []                // 组装部件（如有）
}
```

### 字段说明

| 字段 | 类型 | 说明 | 用途 |
|------|------|------|------|
| `unicode` | string | Unicode 十六进制编码 | 查找符号 |
| `char` | string | 实际字符 | 显示和调试 |
| `glyphId` | integer | 字体内部ID | Android 11+ drawGlyphs() |
| `glyphName` | string | 字形名称 | 调试和文档 |
| `variants` | array | 不同大小的预制变体 | 可拉伸符号 |
| `parts` | array | 可拼接的部件 | 极大符号的拼接 |

## 💻 Android 使用示例

### 1. 加载配置文件

```java
public class MathSymbolConfig {
    private JSONObject config;
    private Map<String, SymbolInfo> symbolMap = new HashMap<>();
    
    public MathSymbolConfig(Context context) throws Exception {
        // 加载 JSON
        InputStream is = context.getAssets().open("latin_modern_math_data.json");
        byte[] buffer = new byte[is.available()];
        is.read(buffer);
        is.close();
        
        String json = new String(buffer, StandardCharsets.UTF_8);
        config = new JSONObject(json);
        
        // 解析符号
        JSONObject symbols = config.getJSONObject("symbols");
        for (Iterator<String> it = symbols.keys(); it.hasNext(); ) {
            String category = it.next();
            JSONObject categorySymbols = symbols.getJSONObject(category);
            
            for (Iterator<String> it2 = categorySymbols.keys(); it2.hasNext(); ) {
                String name = it2.next();
                JSONObject symbolData = categorySymbols.getJSONObject(name);
                
                SymbolInfo info = new SymbolInfo();
                info.name = name;
                info.unicode = symbolData.getString("unicode");
                info.character = symbolData.getString("char");
                info.glyphId = symbolData.getInt("glyphId");
                info.glyphName = symbolData.getString("glyphName");
                
                symbolMap.put(name, info);
            }
        }
    }
    
    public SymbolInfo getSymbol(String name) {
        return symbolMap.get(name);
    }
    
    public int getConstant(String name) throws Exception {
        return config.getJSONObject("constants").getInt(name);
    }
    
    public int getUnitsPerEm() throws Exception {
        return config.getInt("unitsPerEm");
    }
}

class SymbolInfo {
    String name;
    String unicode;
    String character;
    int glyphId;
    String glyphName;
}
```

```java
public void renderSymbol(Canvas canvas, String symbolName, float x, float y) {
    SymbolInfo symbol = config.getSymbol(symbolName);
    
    if (symbol != null) {
        Paint paint = new Paint();
        paint.setTypeface(mathFont);
        paint.setTextSize(fontSize);
        
        // 直接绘制 Unicode 字符

    }
```

```java
public void renderSymbolByGlyphId(Canvas canvas, String symbolName, float x, float y) {
    SymbolInfo symbol = config.getSymbol(symbolName);
    
    if (symbol != null) {
        Paint paint = new Paint();
        paint.setTypeface(mathFont);
        paint.setTextSize(fontSize);
        
        // 使用字形ID绘制（更精确）
        int[] glyphIds = { symbol.glyphId };
        float[] positions = { x, y };
        
        canvas.drawGlyphs(glyphIds, 0, positions, 0, 1, mathFont, paint);
    }
}
```

### 3. 使用数学常量

```java
public float toPixels(int fontUnits, float fontSize) {
    int unitsPerEm = config.getUnitsPerEm();
    return fontUnits * fontSize / unitsPerEm;
}

public void renderFraction(Canvas canvas, float x, float y, float fontSize) {
    // 获取分数线粗细
    int ruleThickness = config.getConstant("FractionRuleThickness");
    float thickness = toPixels(ruleThickness, fontSize);
    
    // 获取分子位置
    int numShift = config.getConstant("FractionNumeratorShiftUp");
    float numY = y - toPixels(numShift, fontSize);
    
    // 获取分母位置
    int denomShift = config.getConstant("FractionDenominatorShiftDown");
    float denomY = y + toPixels(denomShift, fontSize);
    
    // 绘制分数线
    paint.setStrokeWidth(thickness);
    canvas.drawLine(x, y, x + width, y, paint);
}
```

## 🔧 性能优化

### 1. 缓存符号信息

```java
public class SymbolCache {
    private static final Map<String, SymbolInfo> cache = new HashMap<>();
    
    public static SymbolInfo get(MathSymbolConfig config, String name) {
        if (!cache.containsKey(name)) {
            cache.put(name, config.getSymbol(name));
        }
        return cache.get(name);
    }
}
```

### 2. 预加载常用符号

### 3. 批量绘制

```java
@RequiresApi(30)
public void renderMultipleSymbols(Canvas canvas, List<String> symbolNames, 
                                 float[] positions) {
    List<Integer> glyphIds = new ArrayList<>();
    List<Float> glyphPositions = new ArrayList<>();
    
    for (int i = 0; i < symbolNames.size(); i++) {
        SymbolInfo symbol = config.getSymbol(symbolNames.get(i));
        if (symbol != null) {
            glyphIds.add(symbol.glyphId);
            glyphPositions.add(positions[i * 2]);
            glyphPositions.add(positions[i * 2 + 1]);
        }
    }
    
    // 批量绘制
    int[] ids = glyphIds.stream().mapToInt(Integer::intValue).toArray();
    float[] pos = new float[glyphPositions.size()];
    for (int i = 0; i < glyphPositions.size(); i++) {
        pos[i] = glyphPositions.get(i);
    }
    
    canvas.drawGlyphs(ids, 0, pos, 0, ids.length, mathFont, paint);
}
```
